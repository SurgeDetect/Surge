# Issue Drafts

## Issue 1

Title: `Breadth metric still overstates conviction during meme-wide rotation bursts`

Body:
When BONK/WIF/PENGU all light up together the current breadth field looks healthy even if the move is mostly basket rotation. We should distinguish fresh pair-specific buyers from sector-level spillover before calling something `organic_breakout`.

## Issue 2

Title: `Need venue-specific refill tracking instead of one blended refill ratio`

Body:
Some surges look safe because aggregate refill is fine, but the actual lead venue gets thinner while secondary venues keep refilling. That is still fragile for real entries. We should track refill by venue and weight the lead venue more heavily.
